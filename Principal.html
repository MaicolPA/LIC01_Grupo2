¿Qué es la etiqueta canvas?

El elemento HTML CANVAS es una etiqueta que se puede utilizar para dibujar graficos a través de secuencias de comandos (por lo general JavaScript).

Con esta etiqueta podras dibujar graficos, hacer composiciones de fotos o incluso realizar animaciones.

Nuestro Proyecto

Nuestro proyecto se basa en el uso de la etiqueta CANVAS para la creacion de una pizarra en la cual podras dibujar.

Podras elegir un color, un grosor, borrar y guardar tu proyecto. todo esto con la ayuda de la etiqueta CANVAS y el uso de comandos de JavaScript para su utilidad.
